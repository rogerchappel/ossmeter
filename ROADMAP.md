# Roadmap

This roadmap is directional, not a promise.

## Now

- Keep v1 scan local-first and deterministic.
- Improve metric explanations and fixture coverage.
- Validate output shapes with real OSS workspaces.

## Next

- Add optional path filtering for monorepos and archive folders.
- Add richer Markdown report sections for sprint notes.
- Add JSON schema documentation for automation consumers.
- Explore lightweight trend files that users explicitly opt into writing.

## Later

- Optional GitHub enrichment could be considered after v1, but only behind explicit flags.
- Optional historical dashboards may make sense once metric definitions settle.

## Not planned

- Hidden telemetry.
- Remote ranking or productivity moralizing.
- Network calls during default scans.
- Writing into scanned repositories.
