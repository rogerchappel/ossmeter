# Changelog

All notable changes to this project will be documented in this file.

This project follows the [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) format and uses semantic versioning when versioned releases are published.

## [Unreleased]

- Added release-readiness validation for package metadata, CI placeholder cleanup, and package smoke coverage.
### Added

- Local-first `ossmeter scan <workspace>` CLI.
- Workspace discovery for nested git repositories.
- Commit, branch, files-changed, additions, deletions, and dirty worktree metrics.
- Quality signals for tests, CI, README, license, and package metadata.
- Table, Markdown, and JSON output.
- Deterministic fixture generator and smoke command.
- TypeScript public API exports.
